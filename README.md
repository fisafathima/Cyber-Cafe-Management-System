Cyber Cafe Management System

A Python Desktop Application developed using:
- Python
- Tkinter
- MySQL
- Pandas

Features:
- Login System
- Customer Management
- PC Booking
- Billing System
- Reports & Analytics
- Web Scraping
- Networking Notifications
- Generators, Decorators and Closures

Concepts Used:
Data Types, Loops, Conditions, Functions,
Modules, Packages, File Handling,
Exception Handling, OOP, Inheritance,
Polymorphism, Pandas, Tkinter,
Web Scraping, Networking and MySQL.
